# ChoreCorps
### An app in rails for families who get stuff done.

## Development

Use Ruby 2.3.3 and Rails 4.2.3.
To begin developing/tinkering with ChoreCorps, run:

```
$ bundle install
$ rake db:migrate
```

To run your own local development server, run

```
$ rails s
```

You're off to the races!
